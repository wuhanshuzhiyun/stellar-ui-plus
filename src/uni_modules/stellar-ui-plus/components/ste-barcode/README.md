# Barcode 条形码

能够将数字和字母转换生成条形码的组件，支持自定义配色

---$

### 代码演示

#### 基础使用

```html
<template>
    <ste-barcode content="123456abc123" />
</template>
```

#### 自定义颜色

```html
<template>
    <ste-barcode content="123456abc123" foreground="#c11" background="#dad" />
</template>
```
