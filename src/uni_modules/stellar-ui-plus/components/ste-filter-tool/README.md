### API

#### FilterItem

| 事件名     | 说明                                               | 是否必须 |
| ---------- | -------------------------------------------------- | -------- |
| `title`    | 选择项显示的文字                                   | 是       |
| `multiple` | 是否多选, 在更多筛选数据中需要用此字段标识是否多选 | -        |
| `children` | 更多筛选数据选项组数据, 若不是则不需要             | -        |
